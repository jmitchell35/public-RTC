# Frontend (Next.js)

The UI lives under `Front/app/` (Next.js app router).

## Quick start

```bash
cd Front/app
npm install
npm run dev
```

## Environment

- `BACKEND_URL` or `NEXT_PUBLIC_BACKEND_URL`
  - Example: `http://localhost:3001`

## Docs

See `Front/app/README.md` and the READMEs inside `Front/app/app`, `Front/app/components`, and `Front/app/lib` for more details.
