# Frontend lib

Shared helpers for API calls, server actions, and WebSocket types.

Highlights:

- `backend.ts` / `backend-url.ts` : backend URL resolution and fetch
- `actions.ts` : auth actions
- `friend-actions.ts` / `dm-actions.ts` : friend + DM flows
- `ws.ts` / `ws-events.ts` : WebSocket client and types
- `types.ts` : shared TypeScript models
